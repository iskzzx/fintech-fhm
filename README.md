# Fintech FHM

**Fintech FHM** é um projeto de interface web para gerenciamento de hábitos financeiros, desenvolvido como atividade avaliativa na FIAP. Construído com HTML, CSS e Tailwind CSS, o projeto apresenta quatro telas: Dashboard com saldo, métricas e gráfico de gastos; tela de Transferência com seleção de valor e destinatários; histórico de Transações com filtros e paginação; e página de Cadastro. Layout responsivo com suporte a dispositivos mobile.

## Tecnologias

- HTML5
- CSS3
- Tailwind CSS (via CDN)

## Páginas

- `index.html` — Dashboard
- `transfer.html` — Transferir
- `transactions.html` — Transações
- `signup.html` — Criar conta

## Como executar

Basta abrir o arquivo `index.html` diretamente no navegador.
